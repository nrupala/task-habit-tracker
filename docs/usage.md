# User Guide

## Core Workflow

Add task → Template or manual

Mark complete → Tracks time + streaks

Recommendations → Do highest score first

Export CSV weekly → Excel analysis

Backup monthly → Drive sync


## Keyboard Shortcuts
Ctrl+Enter → Save task
Esc → Clear form


## Calendar Legend
Light Blue → 1 habit completed
Green → 2 habits
Dark Green → 3+ habits
Blue Ring → Today


## Recommendation Scores

Habit: +40
Missed today: +15
Overdue: +30
Due tomorrow: +25
Quick win (<15m): +15
